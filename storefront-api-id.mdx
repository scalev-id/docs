Bagian Storefront API terdiri dari dua halaman:

- Mulai dari [Pendahuluan](/storefront-api-id/introduction) untuk mengatur Storefront API dan memahami alur checkout browser-only.
- Baca [Auth](/storefront-api-id/auth) untuk Storefront API key, guest cart token, customer JWT, reset link, CORS header, dan rate limit.
