title: "Edit After Submit Payment Instruction Page"
description: "You could add text variables to display dynamic data such as your bank transfer account list inside the payment instruction description..."
---
On your Scalev checkout page in your landing page, if the [after submit setting](https://tutorial.scalev.id/submite) points to the Payment Instruction Page, then:

**Before the latest update:**

You could add text variables to display dynamic data such as your bank transfer account list inside the payment instruction description.

**After the latest update:**

You **CANNOT** add any dynamic text variables inside the payment instruction description anymore.

You can only write plain description text without dynamic text variables.

**What to watch out for now:**

If your checkout landing page Order Form component uses the Payment Instruction Page and there are text variables in the payment instruction description, edit/remove those dynamic text variables.

Because Scalev now displays those variables automatically.
