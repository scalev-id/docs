To add a TikTok Pixel to Scalev, follow the steps below from start to finish:

### 1. Log in to your TikTok Ads Account

Using the Business Center access you have.

Click this login page link: [https://ads.tiktok.com/i18n/login](https://ads.tiktok.com/i18n/login)

![How to Add TikTok Pixel 1](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-1-826x1024.png)

Login page for TikTok Ads Account

### 2. Select the active ad account you want to use

After entering the Business Center page, click menu Accounts → All Accounts → Select one of your active ad accounts

### 3. Accounts

Click View in the Action column on the right

![How to Add TikTok Pixel 2](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-2-1024x641.png)

Select an ad account to create a new pixel

### 4. Ads Manager

Click the Ads Manager button to open the ad account page.

![How to Add TikTok Pixel 3](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-3-1024x544.png)

Button to go to Ads Manager

### 5. Events

On the Ads Manager page, click menu Tools → Events

![How to Add TikTok Pixel 4](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-4-1024x545.png)

TikTok Ads Account page

### 6. Connect data source

On the Events Manager page, click the **+ Connect data source** button

![How to Add TikTok Pixel 5](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-5-902x1024.png)

Button to create a new TikTok pixel

### 7. Add your website

Enter your website address.

Example:

demo.lpngebut.com

Enter it without the **https://** prefix as it is already filled in automatically.

Click Next.

![How to Add TikTok Pixel 6](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-6-1024x704.png)

Website address input popup

### 8. Choose data connection method

Select **Manual setup** then click the Next button.

![How to Add TikTok Pixel 7](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-7-1024x738.png)

Pixel method popup

### 9. Choose a manual connection method

Select the one recommended by TikTok, which is: TikTok Pixel + Events API.

Click Next.

![How to Add TikTok Pixel 8](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-8-1024x796.png)

TikTok pixel type selection popup

### 10. Create pixel

Give your pixel a name.

The name is up to you.

Example:

Demo Pixel TikTok

![How to Add TikTok Pixel 9](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-9-1024x730.png)

New pixel name popup

### 11. Manage configurations

Note that in the previous step, step 2 "Install base code" — just skip that.

Go directly to step 3.

Enable **Enable Automatic Advanced Matching (Recommended)**.

![How to Add TikTok Pixel 10](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-10-1024x744.png)

Enable Automatic Advanced Matching popup

### 12. Implement Events API

Note that in the previous step, steps 4, 5, and 6 should be skipped by clicking Next repeatedly.

For step 7, click the **Generate access token** button.

![How to Add TikTok Pixel 11](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-11-1024x742.png)

Generate access token popup

After clicking, an access token code like this will appear:

![How to Add TikTok Pixel 12](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-12-1024x399.png)

Access token code

Save and secure the code somewhere safe first.

A Notepad, for example.

Don't forget to click Next.

### 13. Testing

For the testing stage, you can skip it for now.

A dedicated tutorial for testing this TikTok Pixel will be created later.

Click the Finish button.

![How to Add TikTok Pixel 10a](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-10a-1024x728.png)

TikTok Pixel testing popup

### 14. Data sources

Return to the main page of Events Manager.

Click Data sources.

Copy your new TikTok Pixel code.

Save and secure it somewhere.

To a Notepad, for example.

![How to Add TikTok Pixel 13](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-13-1024x629.png)

Copy your TikTok Pixel code

### 15. Settings

Go to the Scalev dashboard then navigate to menu Settings → Analytics.

Select TikTok Pixel in the Analytics dropdown.

Then click **+ Add Analytics**

![How to Add TikTok Pixel 14](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-14-1024x505.png)

Add TikTok Pixel Analytics

### 16. Add Analytics

A popup like the one below will appear.

Fill in the pixel data details that were previously created in the TikTok Business Center

![How to Add TikTok Pixel 15](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-15-730x1024.png)

New TikTok Pixel data input popup

Don't forget to click the Save button.

### 17. Done

If it has been successfully added, the display will look like this.

![How to Add TikTok Pixel 16](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-16-1024x399.png)

TikTok Pixel successfully added to Scalev

**NOTE:**

Please note that adding this new TikTok Pixel, even though it is complete, has **NOT been INSTALLED** automatically on your Landing Page.

So it is not yet ready for use.

To add the TikTok Pixel to your LP, please follow this tutorial:
