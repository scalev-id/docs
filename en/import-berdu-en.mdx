---
title: "Exporting and Importing Customers from Berdu to Scalev (Including Full Order History)"
description: "Log in to Berdu, then go to Apps → Data Exporter."
---
Log in to Berdu, then go to Apps → Data Exporter.

<figure>

![Impor Pelanggan Berdu 1](https://tutorial.scalev.id/wp-content/uploads/2024/08/impor-pelanggan-berdu-1-1024x750.jpeg)

<figcaption>

Open Berdu's customer data export feature via Data Exporter

</figcaption>

</figure>

On the Data Exporter page, select Export Customers then click Download.

Then go to Scalev, navigate to the Customer page, and click the upload (box with upward arrow) button.

<figure>

![Impor Pelanggan Berdu 2](https://tutorial.scalev.id/wp-content/uploads/2024/08/impor-pelanggan-berdu-2-1024x750.jpeg)

<figcaption>

Open the Upload Customer Data feature

</figcaption>

</figure>

A new dialog will appear. Click the Download Template link.

<figure>

![Impor Pelanggan Berdu 3](https://tutorial.scalev.id/wp-content/uploads/2024/08/impor-pelanggan-berdu-3-1024x639.png)

<figcaption>

Download the template file from Scalev

</figcaption>

</figure>

Open the customers\_template.csv file you just downloaded from Scalev using a spreadsheet app like Ms. Excel or Google Sheets.

<figure>

![Impor Pelanggan Berdu 4](https://tutorial.scalev.id/wp-content/uploads/2024/08/impor-pelanggan-berdu-4-1024x473.png)

<figcaption>

Scalev customer document template

</figcaption>

</figure>

The CSV file has 4 default Scalev columns:

1. names

3. phone

5. emails

7. date\_of\_birth

Then open the CSV file downloaded from Berdu as well.

<figure>

![Impor Pelanggan Berdu 5](https://tutorial.scalev.id/wp-content/uploads/2024/08/impor-pelanggan-berdu-5-1024x331.png)

<figcaption>

Berdu customer document template

</figcaption>

</figure>

The Berdu CSV file typically contains columns that can be mapped to Scalev's columns, such as:

1. First Name

3. Last Name

5. Phone

7. Email

Copy the data from the matching columns into the corresponding Scalev columns. For example: paste the data from the First Name column into names, Phone into phones, and so on.

The emails column can be left empty if there's no data for it.

Once all the Berdu data is copied into the customers\_template.csv file, save the changes in your spreadsheet app.

Go back to the Customer → Upload Customer Data page in Scalev.

Click Choose File, select the customers\_template.csv file, then click Upload.

<figure>

![Impor Pelanggan Berdu 6](https://tutorial.scalev.id/wp-content/uploads/2024/08/impor-pelanggan-berdu-6.png)

<figcaption>

Upload Customer Data in Scalev

</figcaption>

</figure>

If the upload succeeds, the customer data import from Berdu to Scalev is complete.
