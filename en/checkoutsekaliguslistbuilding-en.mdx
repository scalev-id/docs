How to Create an Order Form for Email List Building.

The goal of this tutorial is to configure Scalev's Order Form component for **Email List Building.**

You can see a demo at one of our users' sites here: [https://agankhalid.net/checkoutsekaliguslistbuilding](https://agankhalid.net/checkoutsekaliguslistbuilding)

⇣ 

To create an order form like that, follow this tutorial.

- First you need to create a dedicated landing page for list building.

- Then add an order form component.

- Select the product for list building (make sure it's a digital product).

For the order form settings, follow these steps:

1\. Clear the **Form Title**.  
2\. Uncheck **Show Input Label**.

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/1.png)

3\. Disable/Turn off the **Telephone** field.

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/2.png)

4\. Make sure the **Email** field is ON

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/3.png)

5\. Enable/Turn on the **Payment** field.  
6\. Check **Hide Payment**.  
7\. Enable/Turn on **Bank Transfer.**

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/4.png)

8\. For the **Submit Button** field, fill it in as needed.

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/6.png)

9\. Don't forget to Disable/Turn off **Summary**.

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/5.png)

Once configured, your Scalev order form should look like this:

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/7-401x1024.png)

Don't forget to click the Save button at the top of the landing page. Done.

Your order form is now ready to use for email list building.
