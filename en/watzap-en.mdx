WatZap is a WhatsApp marketing tool that lets users send messages to many numbers efficiently.

Key features:

- Send Unlimited Messages

- Smart Auto Reply (Chatbot)

- Fully Customizable Message

- Attach Image on Broadcast Message

- Smart "Perfect Timing" Broadcasting

- Custom Broadcast Scheduling

- etc.

WatZap also integrates with various systems and provides an API for further customization.

Including webhook integration, which Scalev also supports.

The integration here is limited to webhook only.

Not a _deep integration_ like [Starsender](https://tutorial.scalev.id/starsender/ "How to Integrate Starsender with Scalev (For WhatsApp CS Auto Follow-Up When Orders Come In)") or [Woowa](https://tutorial.scalev.id/woowa/ "How to Integrate Woowa with Scalev (For WhatsApp CS Auto Follow-Up When Orders Come In)") which already have dedicated settings inside Scalev.

With the webhook you can connect Scalev with WatZap.

So you can set up a WhatsApp follow-up template exactly like setting it up in Scalev.

But the configuration is inside WatZap.

To integrate Scalev with WatZap, follow the tutorial on their official website at the link below:

[https://docs.watzap.id/help/integrations/integrasi-dengan-scalev.id](https://docs.watzap.id/help/integrations/integrasi-dengan-scalev.id)
