### 3\. Integrate GTM with Scalev

To do this, log in to your Scalev dashboard then follow these steps:

- Click the **Setting** page

- Click the dropdown next to the **Analytics** heading, then select **Google Tag Manager**

- Click **\+ Add Analytics**

<figure>

![](https://tutorial.scalev.id/wp-content/uploads/2024/05/Screenshot-2024-05-10-at-22.05.04.png)

<figcaption>

Enter the Name and Container ID of the GTM you created

</figcaption>

</figure>

- GTM Name can be anything you like

- **Container ID**: enter the GTM ID you created, e.g.: GTM-ABCD1234

<figure>

![](https://tutorial.scalev.id/wp-content/uploads/2024/05/SCR-20240510-tihm-1024x337.png)

<figcaption>

Example Container ID for GTM — copy this text into Scalev Analytics

</figcaption>

</figure>

GTM is now integrated with Scalev.

To activate it, make sure every new landing page you create in Scalev uses the GTM analytics you've installed.

<figure>

![](https://tutorial.scalev.id/wp-content/uploads/2024/05/SCR-20240510-thhg-595x1024.png)

<figcaption>

Select the GTM connected to your Scalev account

</figcaption>

</figure>

Save the landing page when you're done.

Your Google Ads conversion setup for Scalev is now complete and you're ready to run Google ads.
