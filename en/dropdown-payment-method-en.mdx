---
Scalev form now has a feature to make the payment method selection a dropdown.

Dropdown here means a button-like component that when clicked it collapses.

Then the list of banks you enabled appears after it's clicked.

It looks roughly like this:

<figure>

![Collapsible Payment Method](https://tutorial.scalev.id/wp-content/uploads/2024/12/collapsible-payment-method.gif)

<figcaption>

Reveal payment method options with a single click

</figcaption>

</figure>

When this feature is enabled, your payment method options are hidden first.

Until visitors click the dropdown, then all options are revealed.

This feature is useful for temporarily hiding the payment method list in the form.

Especially if you use more than 3 payment methods — without a dropdown, the form looks cluttered.

That can cause distraction when filling out the form.

So the dropdown feature is there to help potential buyers stay focused on the checkout form.

To enable it, follow the steps below:

1. Open the landing page that contains the Order Form component

3. Click the Order Form component, then find the Payment option

5. Click the Payment option and make sure the toggle is active

7. Check Initial View as Dropdown  
      
    ![Dropdown Payment Method](https://tutorial.scalev.id/wp-content/uploads/2024/12/dropdown-payment-method-374x1024.png)  
    

## Events When Payment Methods are Expanded

This dropdown button, like a regular button, can have an event attached — Standard Events or Custom Events — to track the number of interactions in your ads.

This event triggers when the dropdown button is clicked.

<figure>

![Dropdown Payment Method 2](https://tutorial.scalev.id/wp-content/uploads/2024/12/dropdown-payment-method-2.png)

<figcaption>

Select Event When Button is Clicked

</figcaption>

</figure>

To add an event:

- Click the **Events When Payment Methods are Expanded** option

- Then select the event you want

- Or type manually if you want to enter a custom event.

Once done with the settings, don't forget to Save & Publish your landing page.
