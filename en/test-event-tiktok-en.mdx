Before proceeding, make sure you've completed the instructions in the following articles:

1. [How to Add a TikTok Pixel to Scalev](https://tutorial.scalev.id/tambah-pixel-tiktok/ "Tutorial Cara Menambahkan Pixel TikTok ke Scalev")

3. [How to Set Up TikTok Pixel Events on Your Scalev Landing Page](https://tutorial.scalev.id/setting-pixel-tiktok/ "Tutorial Cara Memasang Event Pixel TikTok di Landing Page Scalev")

If done, you can now start testing the TikTok pixel installed on your Scalev landing page.

Follow the steps below.

### 1\. Data sources

Open the TikTok Events Manager page.

In the left menu, click **Data sources**.

Select the pixel you want to test.

<figure>

![Testing Pixel Tiktok 1](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-1-1024x596.png)

<figcaption>

TikTok Events Manager homepage

</figcaption>

</figure>

### 2\. Test Server Events

On the pixel page you selected, click the Test Events tab.

Check the **Test Server Events** section on the left.

Copy the testing code shown in Step 2.

<figure>

![Testing Pixel Tiktok 2](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-2-795x1024.png)

<figcaption>

Copy the testing code

</figcaption>

</figure>

### 3\. Testing Code

Log in to your Scalev dashboard, then go to Settings → Analytics → TikTok Pixel.

Select the TikTok pixel you've installed, then click the three dots on the right.

Click View / Edit.

<figure>

![Testing Pixel Tiktok 3](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-3-1024x436.png)

<figcaption>

Edit TikTok pixel in Scalev Analytics settings

</figcaption>

</figure>

Enter the testing code you copied in the previous step.

![Testing Pixel Tiktok 4](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-4-745x1024.png)

Click Save.

### 4\. Landing Page Link

Go to the Landing Page menu in your Scalev dashboard.

Select and copy the landing page link you want to test.

Example:

demo.lpngebut.com/landing-page-baru

### 5\. Generate QR code

Go back to the TikTok Events Manager page.

In the Test Events tab, paste your landing page link in the box provided.

Then click Generate QR Code.

<figure>

![Testing Pixel Tiktok 5](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-5-795x1024.png)

<figcaption>

Generate a QR Code for your TikTok landing page

</figcaption>

</figure>

A QR code image will appear.

### 6\. Scan QR Code

Scan the QR code using the TikTok app on your phone.

Open the TikTok app.

Check your TikTok account profile.

Click the hamburger menu icon (three lines) in the top right.

<figure>

![Testing Pixel Tiktok 8](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-8-473x1024.jpg)

<figcaption>

Your TikTok profile page

</figcaption>

</figure>

A popup appears — click **My QR Code**.

<figure>

![Testing Pixel Tiktok 9](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-9-473x1024.jpg)

<figcaption>

Access the scan via TikTok's built-in browser

</figcaption>

</figure>

In the top right, click the QR scan icon.

<figure>

![Testing Pixel Tiktok 10](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-10-473x1024.jpg)

<figcaption>

QR Scan icon in the TikTok app

</figcaption>

</figure>

Your phone camera opens — scan the QR code for your landing page generated in step 5.

A page like the image below will appear.

![Testing Pixel Tiktok 11](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-11-473x1024.png)

Click the Open button.

### 7\. Testing Event

Once your landing page is open inside the TikTok app, test your landing page all the way through.

Scroll to the bottom.

Click the button that leads to the form page (if any).

Fill out the checkout form.

Click Submit.

### 8\. Event Activity

Once you've finished testing your landing page in the previous step, go back to the TikTok Events Manager page.

Check the Event Activity section.

<figure>

![Testing Pixel Tiktok 7](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-7-1024x555.png)

<figcaption>

List of events triggered during testing

</figcaption>

</figure>

If the pixel event names you installed on your landing page appear —

Congratulations!

Your TikTok event testing is correct.

Your TikTok pixel setup is successful and ready for advertising on TikTok.

**NOTE**:

If there's a yellow warning icon as shown in the Event Activity screenshot above, you can ignore it.

If you want to fix it, first check the warning to see what needs to be corrected.
