To create a custom audience, you need to prepare data such as email addresses, phone numbers, and more to upload to Facebook Ads. Here are the format requirements you need to meet before uploading:

- At minimum, include at least one of: email, phone number, mobile advertiser ID, Facebook app user ID, Facebook page user ID, first name, or last name.

- Include the country code with phone numbers. If your buyers are in Indonesia, use (+62).

- If you have names, place first and last names in separate columns.

- Make sure the data is saved in ._csv_ format.

The more data you can provide (optional data like location, age, gender, etc.), the better. Facebook can more easily match the uploaded data with data on Instagram/Facebook, making your custom audience far more accurate.

Even if you only operate in certain countries, it's a good idea to include the country data to help Facebook match it with data from around the world.

If you have value data from your uploaded data (such as the total amount earned from a buyer), select **_Yes_**. Adding value data lets you create a lookalike audience and target people based on their purchasing power.

If you don't have value data, select **_No_**. Then check the checkbox at the bottom of the page to agree to Meta's terms, and click **_Next_**.

On the next page, you can upload your prepared data by clicking the upload file section. It's important to give your audience data a name so it's easy to remember and use.

You can also add additional notes by clicking **_Add Description_**. Once the data is uploaded, click **_Next_**.

<figure>

![Image](https://tutorial.scalev.id/wp-content/uploads/2025/12/image-5.png)

On the next page, Facebook will check the data you're about to upload to ensure all the information Facebook needs to create a custom audience is present and each criterion has been matched.

If there are any issues, you can see more details in the **_action needed_** section. To continue, click **_Import & Create_**.

![Image](https://tutorial.scalev.id/wp-content/uploads/2025/12/image-6.png)

</figcaption>
