To verify that the dataset (pixel) events installed on your landing page are running correctly, they need to be tested first via Events Manager in Meta BM.

Follow the guide below.

1\. Go to your Business Manager (BM) homepage in Meta, then open the Events Manager (Pengelola Peristiwa) page

![Events Manager 1 Min](https://tutorial.scalev.id/wp-content/uploads/2024/07/events-manager-1-min-1024x612.png)

2\. On the Events Manager page, check the left sidebar then click Data Sources (Sumber Data)

![Events Manager 2 Min](https://tutorial.scalev.id/wp-content/uploads/2024/07/events-manager-2-min.png)

3\. Once on the Data Sources page:

- Select the pixel to test

- Click the Test events (Uji peristiwa) tab

- Copy the test code

![Events Manager 7 Min](https://tutorial.scalev.id/wp-content/uploads/2024/07/events-manager-7-min-1024x612.png)

4\. Then open the Scalev Analytics settings page at Setting > Analitik then click the pixel you're using

![Events Manager 3 Min](https://tutorial.scalev.id/wp-content/uploads/2024/07/events-manager-3-min-1024x612.png)

5\. The Analytics Account Detail dialog appears — paste the test code from Events Manager into the Testing Code field then Save

![Events Manager 4 Min](https://tutorial.scalev.id/wp-content/uploads/2024/07/events-manager-4-min-748x1024.png)

After saving those settings, open your Scalev landing page in Chrome/Edge browser.

Testing landing pages with Safari is not recommended.

Test your landing page funnel from opening the page to submitting the form.

When the order is received in Scalev, change the order status to Paid.

Then check the Events Manager page in the "Peristiwa yang Diterima" section.

![Events Manager 5 Min](https://tutorial.scalev.id/wp-content/uploads/2024/07/events-manager-5-min-1024x737.png)

If all the events in your funnel appear through the Purchase event, your pixel testing is **CORRECT**.
