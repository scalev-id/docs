To strengthen Scalev's system security following a recent "John Wick" attack, we've updated the search feature with various new restrictions.

Previously, you could search for order or customer data by typing a name.

For now, there are several new rules to follow for more effective searches.

This tutorial will guide you through the new way to search for customers and orders effectively.

## Order Search

Scalev's order search system now applies specific rules based on the format of input you provide.

Here's the guide:

### 1\. Search by Email

How to:

- Enter the full email address containing the "**@**" character

- The system will search using exact match.

Example:

```
customer@contoh.com
fulan@gmail.com
admin@tokoonline.co.id
```

**NOTE:**

- The email must be typed in full and correctly.

- Partial search is not supported for email.

- Make sure there are no spaces at the start or end.

### 2\. Search by Phone Number

How to:

- Enter the phone number in a common format.

- Partial search is supported.

Example:

```
08123456789 (full number)
081234 (partial search)
628123456789 (international format without +)
+62 812 345 6789 (international format with + and spaces)
+62-812-345-6789 (international format with - and +)
```

- Just enter the first few digits for broader results.

- Special characters like + (plus), - (hyphen), and spaces are allowed.

- Partial search is not supported.

## Customer Search

To search for customer data on the Customer page, you can:

- Search by email (exact match).

- Search by phone number (partial search supported).

The way email and phone number search works here is the same as described above for Order search.

**NOTE:**

Customer search currently cannot be performed by entering a customer name.

## Best Practices

### ✅ **DO**

- **Verify** input format before searching.

- **Use** partial search for phone numbers if you're unsure of the full number.

- **Copy-paste** exact match data for order IDs, receipt numbers, and external IDs when possible.

- **Remove** unnecessary spaces at the start and end of your input.

### ❌ **DON'Ts**

- **Don't** mix formats in a single search (e.g., email + phone number).

- **Don't** add special characters to phone number searches, _except_ + and -.

- **Don't** use partial search for email, receipt number, order ID, or external ID.

- **Don't** ignore case sensitivity in exact match searches.

## Troubleshooting

### Common Issues and Solutions

1\. Email search returns no results

- Make sure the email is typed in full with "@"

- Check the spelling of the email address.

- Try an alternative email for the customer.

If the issue persists, try another method. Scalev's backend may be experiencing an issue.

2\. Phone number search is inaccurate

- Make sure only numbers are used.

- Try different formats (with/without country code).

- Use partial search with the first digits.

3\. Receipt/order ID not found

- Double-check the exact spelling.

- Make sure there are no extra spaces.

- Verify case sensitivity (uppercase/lowercase).
