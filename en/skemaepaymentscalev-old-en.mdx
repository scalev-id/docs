## Prolog #2:

If there is a word/amount "FEE" in the video below...
It refers to the FEE charged by Scalev to the Scalev user.

Its name:

**Scalev Fee**.

As for the fee charged by the payment gateway, that's different. ([Fee amount can be found here](https://www.xendit.co/id/biaya "Fee amount can be found here."))

Its name falls under **Transaction Fee**.

\===

E-Payment refers to the payment receiving method from customers to Scalev users.

Which includes:

- VA (virtual account),

- QRIS,

- LinkAja,

- Dana,

- Ovo,

- Shopeepay,

\===

Every transaction that occurs in Scalev has the potential to be charged 2 (two) types of fees.

There are only 2 fees in Scalev, namely:

1. Transaction Fee, costs set by Xendit

3. Scalev Fee, costs set by Scalev

Transaction Fee is the entire cost paid to a **third party** related to the transaction that occurs between **you** and the **customer**.

Example:

- If you transact a digital product using a payment gateway (Xendit)

- If you transact a physical product using the COD payment system (paid to the courier/Ninja)

**Remember, there are only 2 types of fees. No more.**

These 2 fees are only charged for orders using E-Payment.

Orders that do not use E-Payment will **NOT** be charged Transaction Fee or Scalev Fee.

## Transaction Fee Discussion

![Payment fee 1](https://tutorial.scalev.id/wp-content/uploads/2025/03/payment-fee-1-925x1024.png)

Example of Transaction Fee & Scalev Fee

All types of Scalev plans, whether...

- Free

- Lite

- Basic

- Pro

- Ultimate

All are subject to Transaction Fee.

Transaction Fee is charged to users because Scalev uses the Xendit Payment Gateway system.

Applies to all types of transactions using Scalev E-Payment.

Both for Digital Products and Physical Products.

This fee is set by Xendit.

The fee amount depends on the payment method chosen by the user.

Details are in this article: [https://scalev.id/transaction-fee](https://scalev.id/transaction-fee)

## Scalev Fee Discussion

Scalev Fee is a fee charged to users based on the active plan and the LP link used by the user.

This fee is set by Scalev.

Scalev Fee has 2 fee amounts based on the conditions you use on your Scalev account:

### **A. 5% Fee**

This fee applies under 2 conditions:

1. All plans still using Scalev's free subdomain (\*.myscalev.com) on their checkout page link.

3. Scalev Lite plan, even if already using your own Custom Domain.

\===

### **B. 1% Fee**

This fee applies specifically to 3 Scalev plans:

1. Basic Plan

3. Pro Plan

5. Ultimate Plan.

**With the condition:**

Using a **custom domain** on the checkout page link.

Example custom domain: https://cart.kelaskonversi.com/kk-premium

\===
