To add a pixel (now called Datasets in Meta) to Scalev, follow these steps:

1\. First, open Business Manager then go to Business Settings → Data sources → Datasets → Click the pixel ID next to your pixel name

<figure>

![Tambah pixel update](https://tutorial.scalev.id/wp-content/uploads/2024/07/tambah-pixel-update-1024x640.png)

<figcaption>

Copy your pixel code

</figcaption>

</figure>

2\. Open the Scalev dashboard then go to Setting → Analytics → Click + Add Analytics, and a dialog like this will appear:

<figure>

![Tambah Pixel Scalev 3](https://tutorial.scalev.id/wp-content/uploads/2024/07/tambah-pixel-scalev-3-761x1024.png)

<figcaption>

Enter the name and pixel code in the fields marked with a red asterisk

</figcaption>

</figure>

3\. Give the pixel a name then paste the pixel code into the Pixel ID field, then click the Save button

4\. Once saved successfully, the view will look something like this (in this case, multiple pixel IDs have already been added to the Scalev account):

![Tambah Pixel Scalev 4](https://tutorial.scalev.id/wp-content/uploads/2024/07/tambah-pixel-scalev-4-1024x488.png)

To add the Conversion API in your Scalev pixel settings, open this tutorial → [How to Get the Conversion API Token Access](https://tutorial.scalev.id/conversionapi/)
