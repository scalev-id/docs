The Event API pixel type is essential for full integration with the Scalev order form.

Using a SnackVideo Event API pixel, your event funnel will run completely through to the Purchase level, integrated with Scalev's order management system.

Follow these steps to create one.

### 1\. Web Events

Open your SnackVideo ad account homepage at [https://ads.kwai.com/](https://ads.kwai.com/)

Click the Assets menu, then select Web Events.

<figure>

![Membuat Pixel Eventapi 1](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-1-1024x640.png)

<figcaption>

Open the Web Events menu

</figcaption>

</figure>

### 2\. Event API

Once on the Web Events page:

1. Click the Set Up Web button — a new dialog will appear.

3. Give your new pixel a name

5. Select the Event API type

7. Then click Next

<figure>

![Membuat Pixel Eventapi 2](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-2-1024x640.png)

<figcaption>

Set the SnackVideo pixel type

</figcaption>

</figure>

### 3\. OK

On the next page, simply scroll down and click OK.

<figure>

![Membuat Pixel Eventapi 3](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-3-1024x640.png)

<figcaption>

Ignore the instructions on this page

</figcaption>

</figure>

### 4\. Generate Access Token

On the newly created pixel page:

1. Click the Settings tab

3. Click the Generate Access Token button

<figure>

![Membuat Pixel Eventapi 4](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-4-1024x640.png)

<figcaption>

Get the API token from the new pixel

</figcaption>

</figure>

### 5\. Copy Token API

Once the code appears, click Copy to copy this API token.

<figure>

![Membuat Pixel Eventapi 5](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-5-1024x386.png)

<figcaption>

The generated API token code

</figcaption>

</figure>

### 6\. Add Analytics

Then go to your Scalev dashboard to add the new SnackVideo pixel.

1. Click Settings

3. Click Analytics

5. Select SnackVideo Pixel

7. Click Add Analytics

<figure>

![Membuat Pixel Eventapi 6](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-6-1024x640.png)

<figcaption>

Add to Scalev Analytics settings

</figcaption>

</figure>

The Add Analytics dialog appears as shown below.

<figure>

![Membuat Pixel Eventapi 7](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-7-717x1024.png)

<figcaption>

Fill in the new SnackVideo pixel details

</figcaption>

</figure>

Fill in all the required details then click Save.

### 7\. Done

If successfully added, it will appear like this.

<figure>

![Membuat Pixel Eventapi 8](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-8-1024x439.png)

<figcaption>

Scalev Analytics settings page

</figcaption>

</figure>

Your SnackVideo Event API pixel has been successfully added to Scalev.

But you're not done yet — you need to install this new pixel in your landing page settings.

Follow the guide in this article: [How to Set Up SnackVideo Pixel on the Scalev Platform](https://tutorial.scalev.id/setting-pixel-snackvideo/ "Tutorial Cara Setting Pixel SnackVideo di Platform Scalev")
