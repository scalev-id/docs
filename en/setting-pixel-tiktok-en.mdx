---
title: "How to Set Up TikTok Pixel Events on Your Scalev Landing Page"
description: "Follow these steps:"
---
If you've already [added a new TikTok pixel](https://tutorial.scalev.id/tambah-pixel-tiktok/ "How to Add a TikTok Pixel to Scalev"), you now need to install the pixel and the required events on your landing page.

Follow these steps:

### 1\. Landing Page Studio (LPS)

Go to your Scalev dashboard, then navigate to the Landing Page menu and select or create a new landing page.

<figure>

![Cara Setting Pixel Tiktok 1](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-setting-pixel-tiktok-1-1024x408.png)

<figcaption>

Open your landing page in Scalev

</figcaption>

</figure>

### 2\. TikTok Pixel Settings

In the LPS page, click the Settings tab.

Then scroll down until you find the TikTok Pixel section.

Select and install the TikTok pixel you've added to Scalev.

Also set the event that matches your landing page.

<figure>

![Cara Setting Pixel Tiktok 2](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-setting-pixel-tiktok-2-898x1024.png)

<figcaption>

TikTok pixel settings in the landing page

</figcaption>

</figure>

### 3\. Order Form Component

Still on the LPS page.

Click the Content tab, then edit or add the Order Form component.

Click the Gear icon to go to the Order Form component settings.

<figure>

![Cara Setting Pixel Tiktok 3](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-setting-pixel-tiktok-3-953x1024.png)

<figcaption>

Order Form component settings

</figcaption>

</figure>

Scroll down until you find the TikTok pixel event section.

Enter a standard TikTok event, for example: InitiateCheckout.

<figure>

![Cara Setting Pixel Tiktok 4](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-setting-pixel-tiktok-4-884x1024.png)

<figcaption>

TikTok event in the Order Form component

</figcaption>

</figure>

This event fires when a visitor submits the form.

### 4\. Repeat

Repeat the steps above for your other active landing pages as needed for your funnel.

### 5\. Done.

At this point you've successfully set up the TikTok pixel with its events installed on your active landing pages.

Next, you need to test these TikTok pixel events to make sure all settings are correct.

Click the article title below for a guide on testing your TikTok pixel.
