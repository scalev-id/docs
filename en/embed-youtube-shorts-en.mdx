If you want to display YouTube Shorts on your Scalev landing page, you can use the tool above.

Unlike regular YouTube videos, Shorts embed code isn't easily provided directly by YouTube.

To make it easier, we built the tool above so you can generate ready-to-use embed code.

**This tool is built specifically for Scalev users.**

Follow the steps below.

### 1\. Open YouTube

Open your video on the YouTube website or app.

1. Click the Share button on the right of the video

3. Click the Copy Link button

<figure>

![Embed Yt Shorts 1](https://tutorial.scalev.id/wp-content/uploads/2024/12/embed-yt-shorts-1-1024x996.png)

<figcaption>

Copy YouTube Shorts URL

</figcaption>

</figure>

### 2\. Generate Embed Code

Use the tool at the start of this article.

1. Paste your video link in the URL field

3. Click Generate Embed Code — the result appears in the box below.

5. Click the Copy icon to copy the code

![Embed Yt Shorts 2](https://tutorial.scalev.id/wp-content/uploads/2024/12/embed-yt-shorts-2.png "Generate embed code yt shorts")

### 3\. HTML Component

Open Landing Page Studio in your Scalev dashboard.

Then add an HTML component.

Paste the embed code you just copied into this HTML component.

Save & Publish your landing page.

![Embed Yt Shorts 3](https://tutorial.scalev.id/wp-content/uploads/2024/12/embed-yt-shorts-3-992x1024.png)

Done.
