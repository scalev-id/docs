To create a floating button leading to the order form, follow these steps:

- Add a scroll target component above the order form component.

![CleanShot 2026 04 14 at 13.52.21@2x](https://tutorial.scalev.id/wp-content/uploads/2025/05/CleanShot-2026-04-14-at-13.52.21@2x.png)

- Add a button component at the very bottom of your landing page

![CleanShot 2026 04 14 at 13.53.12@2x](https://tutorial.scalev.id/wp-content/uploads/2025/05/CleanShot-2026-04-14-at-13.53.12@2x.png)

- Configure the button component like this:

![CleanShot 2026 04 14 at 13.53.59@2x](https://tutorial.scalev.id/wp-content/uploads/2025/05/CleanShot-2026-04-14-at-13.53.59@2x-411x1024.png)

1. Set Position to **Sticky**

3. Set the button type to **Scroll target**

5. Point the Scroll target to the **Scroll Target** component you placed above the order form.

Once done, **Save and Publish** your landing page.
