---
title: "Storefront API"
description: "Buat storefront custom di browser dengan Scalev sebagai backend commerce."
---

Bagian Storefront API berisi:

- Mulai dari [Pendahuluan](/id/storefront-api-id/introduction) untuk mengatur Storefront API dan memahami alur checkout browser-only.
- Baca [Auth](/id/storefront-api-id/auth) untuk Storefront API key, guest cart token, customer JWT, reset link, CORS header, dan rate limit.
- Gunakan [Checkout dan Payment](/id/storefront-api-id/checkout-payments) untuk shipping options, checkout summary, guest checkout, order payment, dan rendering payment per metode.
