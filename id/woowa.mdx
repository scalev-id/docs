Cara Meng-integrasikan Woowa Ke Scalev

Untuk mengintegrasikan akun Woowa Anda ke akun Scalev Anda, silahkan ikuti petunjuk ini:

1\. Pergi ke menu **Setting.**

2\. Klik **Integrasi**.

3\. Pilih **Integrasi dengan WhatsApp Gateway**

4\. Klik **Tambah Integrasi**  
  
![](https://tutorial.scalev.id/wp-content/uploads/2024/03/CleanShot-2024-03-25-at-07.29.40-1024x565.png)

5\. Pilih Woowa.

6\. Login akun Woowa Anda.

7\. Klik **More** pada kolom **Action**

![](https://tutorial.scalev.id/wp-content/uploads/2024/03/1-1-1024x583.png)

8\. Ketika pilihannya muncul, Klik **Get Key.**

![](https://tutorial.scalev.id/wp-content/uploads/2024/03/2-1-1024x584.png)

Tunggu hingga muncul API Key Woowanya.

9\. Copy License Code & Woowa Key Anda

![](https://tutorial.scalev.id/wp-content/uploads/2024/03/3-1-1024x577.png)

10\. Paste semuanya ke Scalev.

11\. Klik **Simpan**.

![](https://tutorial.scalev.id/wp-content/uploads/2024/03/CleanShot-2024-03-25-at-07.32.26.png)
