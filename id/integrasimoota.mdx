Setiap akun Moota hanya bisa diintegrasikan ke 1 akun Scalev saja.

Untuk melakukan integrasi dengan Moota, silahkan ikuti langkah-langkah dibawah ini:

- Silahkan masuk ke Akun Scalev Anda.

- Klik Menu Setting.

- Kemudian Pilih Integrasi ⇢ Moota.

- Klik Tambah Integrasi Moota

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/CleanShot-2024-04-02-at-08.52.22-1024x583.png)

Nanti akan muncul window untuk Anda memasuka API Token dari Moota seperti ini:

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/CleanShot-2024-04-02-at-08.53.17.png)

Untuk mendapatkan API Token Moota, silahkan:

-  Pergi ke dasboard Moota.

- Klik Menu Integrasi.

- Pilih API Token.

- Masukan Nama Token (Ini bebas).

- Checklist API (WAJIB)

- Lalu Scroll Ke bawah

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/CleanShot-2024-04-02-at-08.55.00-1024x541.png)

- Kemudian klik tombol "Buat Personal API Token".

- Lalu Copy API Tokenya.

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/CleanShot-2024-04-02-at-08.56.26-1024x745.png)

- Kembali ke Scalev Anda.

- Paste-kan API Token yang tadi sudah Anda Copy Sebelumnya ke field Moota Token

- Klik Simpan 

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/CleanShot-2024-04-02-at-08.57.23.png)

Jika berhasil, maka Moota Anda akan muncul seperti ini:

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/CleanShot-2024-04-02-at-09.00.30-1024x571.png)

Sampai disini Anda sudah berhasil integrasikan Moota Ke Scalev.

Langkah selanjutnya adalah mengaktifkan kode unik. Ini WAJIB dilakukan supaya Scalev tidak kebingungan menandai orderan mana yg telah dibayar & mana yg belum.

Untuk mengaktifkan kode unik silahkan ikuti langkah-langkah ini:

- Masuk ke menu Store.

- Pilih toko yg mau diaktifkan kode uniknya (jika Anda memiliki banyak toko, kami wajibkan Anda mengedit semua toko) ⇢ Klik Edit toko.

- Scroll ke bawah sampai menemukan kode unik di sebelah kanan ⇢ Klik Lihat

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/CleanShot-2024-04-02-at-09.01.54-1024x581.png)

- Kemudian ubah pengaturan kode unik dari Non aktif menjadi Aktif.

- Lalu atur batas kode uniknya (kami sarankan atur hingga 999).

- Dan klik Simpan.

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/CleanShot-2024-04-02-at-09.02.39.png)

Langkah terakhir,

Silahkan Scroll kebawah hingga melihat pilihan Cek Transfer dengan Moota.

Kemudian klik Aktifkan Moota.

Dan klik tombol Simpan di sisi kanan atas.

![](https://tutorial.scalev.id/wp-content/uploads/2024/04/CleanShot-2024-04-02-at-09.03.45-1024x579.png)

Selamat Akun Moota Anda sudah berhasil di Integrasikan ke Scalev.

Silahkan coba buat pesanan di toko Anda & lakukan pembayaran via transfer untuk memastikan Cek Transfer dengan Moota telah aktif.

* * *

**Catatan**:

- Biasanya transferan akan terbaca paling cepat 5 - 15 menit, paling lama 1 jam.

- Jika terjadi kendala, silahkan cek seluruh step diatas, apakah ada yang terlewat atau tidak.

- Jika merasa semua sudah sesuai & masih terjadi kendala, silahkan hubungi CS kami.
