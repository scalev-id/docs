Awalnya hanya custom domain yang dapat digunakan dengan [Mailev](https://tutorial.scalev.id/category/mailev/ "Mailev") adalah domain yang sudah tersambung dan ditambahkan di Scalev.

Namun sekarang sudah bisa menambahkan custom domain milik Anda sendiri walaupun tidak ada dalam daftar domain terverifikasi di Scalev.

Ikuti langkah-langkah berikut ini:

Masuk dasbor Scalev

Di dasbor Scalev, klik ikon menu Mailev di sebelah kiri

<figure>

![Mailev identity manual](https://tutorial.scalev.id/wp-content/uploads/2026/02/mailev-identity-manual-1-768x1024.png)

<figcaption>

Klik ikon Mailev

</figcaption>

</figure>

Kemudian Anda akan diarahkan dan login secara otomatis ke Mailev

Di dasbor Mailev, klik dropdown menu lalu klik **Identity**

<figure>

![Mailev identity manual](https://tutorial.scalev.id/wp-content/uploads/2026/02/mailev-identity-manual-2-1024x576.png)

<figcaption>

Menu dropdown Mailev

</figcaption>

</figure>

Di halaman Identity klik tombol **\+ Add Identity**

<figure>

![Mailev identity manual](https://tutorial.scalev.id/wp-content/uploads/2026/02/mailev-identity-manual-3-1024x825.png)

<figcaption>

Klik menu Identity

</figcaption>

</figure>

Muncul dialog **Add New Identity**

<figure>

![Mailev identity manual](https://tutorial.scalev.id/wp-content/uploads/2026/02/mailev-identity-manual-4-1024x400.png)

<figcaption>

Klik tombol Add Identity

</figcaption>

</figure>

Beri nama untuk Identity ini di field **Title**

Lalu, klik teks **Enter manually** untuk memasukkan custom domain Anda secara manual

<figure>

![Mailev identity manual](https://tutorial.scalev.id/wp-content/uploads/2026/02/mailev-identity-manual-5.png)

<figcaption>

Klik Enter Manually

</figcaption>

</figure>

Masukkan domain Anda tanpa awalan http/https di field **Domain**

Selanjutnya ikuti tutorial cara menambahkan Identity di artikel berikut: [https://tutorial.scalev.id/mailev-identity/](https://tutorial.scalev.id/mailev-identity/)
