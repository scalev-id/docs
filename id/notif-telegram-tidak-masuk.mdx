Notif orderan di Telegram tidak masuk padahal sudah integrasi dan hubungkan akun TG? Cara mengatasinya:

## Lakukan Integrasi Ulang

1. **Hapus Akun Telegram.**

Masuk ke menu Settings>Profil>Hapus Akun Telegram

![](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEjryOXB5CG0-Winoxz_q4xyR3izoyOA_mTnVyNQHaTjuEvhh_TVdDXEoX6xzp1WymFA47iLCXIgCor6OwiLY2AKbZm0TnPpLqeLxPwq12E33tAGOPnh4D-ePfygD71TayC_dfhLlmBHqTfNR2fTzjiYRgE0J3xP3VI809_28ruLaUnmAaV3PAum9V-LUgQ/s1600/Z.webp)

2\. **Hubungkan Kembali Akun Telegram**

![](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEjZZKosx_Erq9gJp_eqfAYDK13sBVAeDgQJLXFJljRS1lwqeern-3kdysZ25dKFcRq1C6CBlyXNwbKeJmpbb7y3HJmh0YfWbglfVSH4HX0I59a4FDKhi_qHHUsfYsq0di5Tu9eCyOlZEF72kYr79RMc_SmVv_J-bwGTnoXvI5XFO9mWwfwNnvqYl400cSA/s1600/Z1.webp)

## Uncentang Notifikasi

Masih dimenu Settings>Profil, Klik **Configure Order Notification with Telegram**

![](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEh8QyPFXcl2Q_afEoN2MVCpSnquwlOf1_H63i-m97M3kHGU0qBH-U6W3pzYNCZkMmmbdAr3gEpLtORrKE8UxhL8uavfEFpZ3gGm2nIjSCFbC0kZwLNzGZczNm6bFi4eCBQqBz9ixmTjWRg7jSCWn7WMJdiheIRMbSVHhS4Q6AUBQGepwzlce2mDO_CpEio/s1600/Z2.webp)

Hilangkan **semua centang** pada konfigurasi notifikasi. Lalu klik **save**.

![](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEihtqzrmJ13OV970B_g1SbsGFT3Zfl2k2WDpZV3MCthBfyxUXaQrmd0_lYj9Rme0EsoUpdpAYoFn1CN48LGB0GUItyfjFbkpvo0xryEoWd41G2pa9IybJNZm7BXhPlLIhPbuIvPDI6eZV6kMjvkzGOt_wAaYbM9HsjaD34tHgJaLjZqEXZUnTG3rlfltHk/s1600/Z3.png)

## Test Order 1 (WAJIB)

Setelah melakukan uncentang pada pengaturan notifikasi, silahkan pergi ke Landing page Anda. Lakukan test order. Isi data yang lengkap. Test order ini wajib dilakukan.

Catatan: Untuk test order pertama ini tidak akan ada notifikasi yang terkirim di Telegram. Lanjut.

## Centang Notifikasi

Setelah test order silahkan dicentang kembali bagian pengaturan **Configure Order Notification with Telegram**. Centang semua, lalu **save**.

![](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEhGxGnwUzGLgEpeiO3M3anV95_fdp5JJypEJJ-MDEr_AUs3PZ0Bltxn3TsSoPu8_kQdH6xqWX0VQAUgvjx50mW7S1ryH55heT6GqXTisAinhTTq90mSwaIJGZds2phZ7P0_mXD8JAgRB0oQXNoxIQdItfw04UUr80S3AAX13ullyQFH_lTk9VAwGEdNNug/s1600/Z5.png)

## Selesai

Silahkan test order lagi. Cek notifikasi di Telegramnya.

## **Penting:** Wajib lakukan langkah diatas secara berurutan.
