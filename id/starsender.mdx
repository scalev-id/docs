* * *

- Silahkan menuju menu setting.

- Klik Integrasi.

- Lalu klik Tambah Integrasi WhatsApp Gateway

![Cleanshot 2024 08 07 At 10.19.38](https://tutorial.scalev.id/wp-content/uploads/2024/08/CleanShot-2024-08-07-at-10.19.38-1024x585.png)

- Masukan nama (bebas sesuai kebutuhan Anda).

- Masukan nomor telepon Starsender Anda

- Untuk Service Provider → Pilih Starsender.

![CleanShot 2025 11 27 at 07.56.42@2x](https://tutorial.scalev.id/wp-content/uploads/2024/08/CleanShot-2025-11-27-at-07.56.42@2x.png)

* * *

Untuk Starsender Device API Key,

- Silahkan kembali ke dasboard Starsender.

- Klik Device.

- Copy API Key Device yang ingin diintegrasikan ke Scalev.

![CleanShot 2025 11 27 at 08.01.12@2x](https://tutorial.scalev.id/wp-content/uploads/2024/08/CleanShot-2025-11-27-at-08.01.12@2x-1024x575.png)

- Paste Device API Key yang tadi di copy ke Scalev.

![CleanShot 2025 11 27 at 08.00.42@2x](https://tutorial.scalev.id/wp-content/uploads/2024/08/CleanShot-2025-11-27-at-08.00.42@2x.png)

- Jika berhasil, maka akan muncul keterangan "WhatsApp Gateaway Account has been successfully added" & "Test message successfully sent"

- Artinya Starsender Anda siap digunakan di Scalev.

![Image](https://tutorial.scalev.id/wp-content/uploads/2024/08/image-3.png)
