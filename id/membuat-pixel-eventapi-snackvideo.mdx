Tipe pixel Event API sangat diperlukan agar dapat mengintegrasikan secara penuh dengan form Scalev.

Dengan menggunakan pixel Event API SnackVideo, maka funnel event Anda akan berjalan secara lengkap sampai level Purchase yang terintegrasi dengan sistem manajemen order di Scalev.

Cara membuatnya ikuti langkah-langkah berikut ini.

### 1\. Web Events

Buka halaman depan akun iklan SnackVideo Anda di [https://ads.kwai.com/](https://ads.kwai.com/)

Klik menu Assets, lalu pilih Web Events.

<figure>

![Membuat Pixel Eventapi 1](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-1-1024x640.png)

<figcaption>

Buka menu Web Events

</figcaption>

</figure>

### 2\. Event API

Setelah masuk ke halaman Web Events:

1. Klik tombol Set Up Web, lalu muncul dialog baru.

3. Beri nama untuk pixel baru Anda

5. Pilih tipe Event API

7. Lalu klik Next

<figure>

![Membuat Pixel Eventapi 2](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-2-1024x640.png)

<figcaption>

Mengatur tipe pixel SnackVideo

</figcaption>

</figure>

### 3\. OK

Halaman berikutnya cukup scroll ke bawah lalu klik tombol OK.

<figure>

![Membuat Pixel Eventapi 3](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-3-1024x640.png)

<figcaption>

Abaikan instruksi di halaman ini

</figcaption>

</figure>

### 4\. Generate Access Token

Di halaman pixel yang baru dibuat tadi:

1. Klik tab Settings

3. Klik tombol Generate Access Token

<figure>

![Membuat Pixel Eventapi 4](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-4-1024x640.png)

<figcaption>

Mengambil token API dari pixel baru

</figcaption>

</figure>

### 5\. Copy Token API

Setelah kodenya muncul, klik tombol Copy untuk menyalin token API ini.

<figure>

![Membuat Pixel Eventapi 5](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-5-1024x386.png)

<figcaption>

Kode token API yang telah digenerate

</figcaption>

</figure>

### 6\. Tambah Analitik

Setelah itu pergi ke dasbor Scalev untuk menambahkan pixel baru SnackVideo.

1. Klik Settings

3. Klik Analitik

5. Pilih SnackVideo Pixel

7. Klik Tambah Analitik

<figure>

![Membuat Pixel Eventapi 6](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-6-1024x640.png)

<figcaption>

Tambahkan ke pengaturan Analitik Scalev

</figcaption>

</figure>

Muncul dialog Tambah Analitik seperti di bawah ini.

<figure>

![Membuat Pixel Eventapi 7](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-7-717x1024.png)

<figcaption>

Lengkapi dengan rincian pixel baru SnackVideo

</figcaption>

</figure>

Lengkapi semua data yang diperlukan lalu klik tombol Simpan.

### 7\. Done

Selanjutnya akan tampil seperti ini jika berhasil ditambahkan.

<figure>

![Membuat Pixel Eventapi 8](https://tutorial.scalev.id/wp-content/uploads/2024/12/membuat-pixel-eventapi-8-1024x439.png)

<figcaption>

Halaman pengaturan Analitik Scalev

</figcaption>

</figure>

Pixel Event API SnackVideo Anda sudah berhasil ditambahkan ke Scalev.

Tapi belum selesai, Anda perlu memasang pixel baru tersebut ke pengaturan LP yang akan digunakan.
