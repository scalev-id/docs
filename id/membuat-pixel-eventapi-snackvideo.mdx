Tipe pixel Event API sangat diperlukan agar dapat mengintegrasikan secara penuh dengan form Scalev.

Dengan menggunakan pixel Event API SnackVideo, maka funnel event Anda akan berjalan secara lengkap sampai level Purchase yang terintegrasi dengan sistem manajemen order di Scalev.

Cara membuatnya ikuti langkah-langkah berikut ini.

### 1. Web Events

Buka halaman depan akun iklan SnackVideo Anda di [https://ads.kwai.com/](https://ads.kwai.com/)

Klik menu Assets, lalu pilih Web Events.

<Frame>
  ![Membuat Pixel Eventapi 1 1024x640](/images/membuat-pixel-eventapi-1-1024x640.png)
</Frame>

### 2. Event API

Setelah masuk ke halaman Web Events:

1. Klik tombol Set Up Web, lalu muncul dialog baru.
2. Beri nama untuk pixel baru Anda
3. Pilih tipe Event API
4. Lalu klik Next

<Frame>
  ![Membuat Pixel Eventapi 2 1024x640](/images/membuat-pixel-eventapi-2-1024x640.png)
</Frame>

### 3. OK

Halaman berikutnya cukup scroll ke bawah lalu klik tombol OK.

<Frame>
  ![Membuat Pixel Eventapi 3 1024x640](/images/membuat-pixel-eventapi-3-1024x640.png)
</Frame>

### 4. Generate Access Token

Di halaman pixel yang baru dibuat tadi:

1. Klik tab Settings
2. Klik tombol Generate Access Token

<Frame>
  ![Membuat Pixel Eventapi 4 1024x640](/images/membuat-pixel-eventapi-4-1024x640.png)
</Frame>

### 5. Copy Token API

Setelah kodenya muncul, klik tombol Copy untuk menyalin token API ini.

<Frame>
  ![Membuat Pixel Eventapi 5](/images/membuat-pixel-eventapi-5.png)
</Frame>

### 6. Tambah Analitik

Setelah itu pergi ke dasbor Scalev untuk menambahkan pixel baru SnackVideo.

1. Klik Settings
2. Klik Analitik
3. Pilih SnackVideo Pixel
4. Klik Tambah Analitik

<Frame>
  ![Membuat Pixel Eventapi 6 1024x640](/images/membuat-pixel-eventapi-6-1024x640.png)
</Frame>

Muncul dialog Tambah Analitik seperti di bawah ini.

<Frame>
  ![Membuat Pixel Eventapi 7 717x1024](/images/membuat-pixel-eventapi-7-717x1024.png)
</Frame>

Lengkapi semua data yang diperlukan lalu klik tombol Simpan.

### 7. Done

Selanjutnya akan tampil seperti ini jika berhasil ditambahkan.

<Frame>
  ![Membuat Pixel Eventapi 8 1024x439](/images/membuat-pixel-eventapi-8-1024x439.png)
</Frame>

Pixel Event API SnackVideo Anda sudah berhasil ditambahkan ke Scalev.

Tapi belum selesai, Anda perlu memasang pixel baru tersebut ke pengaturan LP yang akan digunakan.

Panduannya silakan ikuti di artikel ini: [Tutorial Cara Setting Pixel SnackVideo di Platform Scalev](https://tutorial.scalev.id/setting-pixel-snackvideo/ "Tutorial Cara Setting Pixel SnackVideo di Platform Scalev")
