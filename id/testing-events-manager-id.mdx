Untuk memastikan dataset (pixel) yang terpasang di LP Anda events-nya sudah berjalan dengan benar maka perlu diuji terlebih dulu oleh Events Manager di BM Meta.

Caranya silakan ikuti panduan berikut ini.

1\. Masuk ke beranda Business Manager (BM) Anda di Meta, lalu buka halaman Events Manager (Pengelola Peristiwa)

![Events Manager 1 Min](https://tutorial.scalev.id/wp-content/uploads/2024/07/events-manager-1-min-1024x612.png)

2\. Di halaman Events Manager cek bilah sisi kiri lalu klik Data Sources (Sumber Data)

![Events Manager 2 Min](https://tutorial.scalev.id/wp-content/uploads/2024/07/events-manager-2-min.png)

3\. Kalau sudah di halaman Data Sources:

- Pilih pixel yang akan dites

- Klik tab Test events (Uji peristiwa)

- Salin kode test

![Events Manager 7 Min](https://tutorial.scalev.id/wp-content/uploads/2024/07/events-manager-7-min-1024x612.png)

4\. Setelah itu buka halaman pengaturan Analitik Scalev di Setting > Analitik lalu klik pixel yang Anda pakai

![Events Manager 3 Min](https://tutorial.scalev.id/wp-content/uploads/2024/07/events-manager-3-min-1024x612.png)

5\. Muncul dialog Detail Akun Analitik, tempelken kode test dari Events Manager ke field Kode Testing setelah itu Simpan

![Events Manager 4 Min](https://tutorial.scalev.id/wp-content/uploads/2024/07/events-manager-4-min-748x1024.png)

Setelah menyimpan pengaturan tersebut langkah selanjutnya adalah membuka LP Anda di Scalev dengan browser Chrome/Edge.

Tidak direkomendasikan mengetes LP dengan browser Safari.

Silakan test funnel LP Anda mulai dari membuka halaman sampai submit form.

Ketika order sudah masuk ke Scalev, ubah status orderan menjadi Paid.

Kemudian perhatikan halaman Events Manager di bagian "Peristiwa yang Diterima".

![Events Manager 5 Min](https://tutorial.scalev.id/wp-content/uploads/2024/07/events-manager-5-min-1024x737.png)

Jika semua event yang ada di funnel Anda sudah muncul sampai event Purchase, maka pengujian pixel Anda sudah **BENAR**.

NOTE:

Jika tampilan BM Anda berbahasa Inggris, lihat versi bahasa Inggrisnya di sini → [https://tutorial.scalev.id/testing-events-manager/](https://tutorial.scalev.id/testing-events-manager/)
