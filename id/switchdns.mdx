Sebelum melakukan _Switch to New Confirguration_ Custom Domain Silahkan pastikan 2 hal ini:

1. Pastikan DNS manager Anda support terhadap DNS type CNAME, kalau belum bisa gunakan Cloudflare. Tutorialnya → [https://tutorial.scalev.id/cloudflare/](https://tutorial.scalev.id/cloudflare/)

3. Matikan iklan terlebih dahulu, karena selama proses pemindahan custom domain Anda tidak akan bisa diakses.
