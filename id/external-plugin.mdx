Dulu, Scalev memiliki sifat yang bisa dibilang "egois" dalam berinteraksi dengan tool atau software lain:

- Scalev → Tool lain: Bisa trigger aksi ✅

- Tool lain → Scalev: Tidak bisa trigger aksi ❌

Ibarat orang yang maunya didengar tapi tidak mau mendengar.

## Solusi Terkini

Kini Scalev sudah berubah menjadi platform yang lebih kolaboratif:

- Mau mendengar input dari tool lain ✅

- Mau mengirim data ke tool lain ✅

- Siap berintegrasi dengan platform manapun ✅

## Cara Integrasi untuk End User

Jika Anda pengguna platform/aplikasi tertentu yang ingin melakukan integrasi dengan Scalev:

### 1\. Request ke platform Anda

Tanyakan kepada mereka untuk mengembangkan aplikasi integrasi dengan Scalev.

### 2\. Tunggu implementasi

Tergantung kesediaan platform untuk mengalokasikan resource dalam mengerjakan implementasi tersebut.

### 3\. Connect di Scalev

Setelah diimplementasikan maka kedua platform dapat saling berkomunikasi data.

Silakan akses platform tersebut di menu **Settings** > **Developers** > **Apps** untuk melakukan proses integrasi.

## Panduan untuk Developer

Bagi developer yang ingin mengintegrasikan platform mereka dengan Scalev:

### 1\. Pelajari Dokumentasi API

Link: [https://docs.scalev.id/auth.html](https://docs.scalev.id/auth.html)

### 2\. Konsultasi Teknis

Gabung di grup Telegram Scalev khusus developer di "**Coding Area!**": [https://developers.scalev.id/](https://developers.scalev.id/ "https://developers.scalev.id/")

### 3\. Implementasi

Jika Anda developer yang kompeten, kedua resource di atas sudah cukup untuk memahami cara implementasinya.

## Kesimpulan

Scalev kini menyediakan peluang integrasi untuk aplikasi pihak ketiga melalui fitur Apps yang dapat diakses di dasbor.

Dokumentasi dan support channel sudah tersedia.

Selanjutnya tinggal menunggu platform pihak ketiga yang memutuskan untuk memanfaatkan peluang integrasi ini atau tidak.
