Default events adalah event pixel bawaan yang sudah disetel secara otomatis oleh Scalev di Landing Page Studio.

Default events Scalev itu ada empat, yaitu:

1. ViewContent

3. AddToCart

5. InitiateCheckout

7. Purchase

## Event ViewContent

ViewContent sudah terpasang secara otomatis saat Anda pertama kali membuat Landing Page baru tipe apapun di Scalev.

Posisinya terletak di Landing Page Studio → Pengaturan → Pengaturan Analitik

## Event AddToCart

AddToCart sudah terpasang secara otomatis jika Anda membuat landing page baru tipe Checkout Page.

Untuk memastikan bahwa event AddToCart terpasang secara otomatis silakan Anda cek pengaturan LP nya.

Cek di Pengaturan → Analytics Setting.

Scroll-scroll dulu ke bawah untuk menemukan lokasi tersebut.

<figure>

![Default Events 8](https://tutorial.scalev.id/wp-content/uploads/2024/07/default-events-8.png)

<figcaption>

Pengaturan event AddToCart di LP Scalev

</figcaption>

</figure>

Facebook Pixel ID akan terisi secara otomatis jika Anda sudah menambahkan pixel FB di pengaturan Analitik Scalev.

Event Saat Landing Page Terbuka akan diisi otomatis dengan AddToCart jika Anda membuat LP baru dengan tipe Checkout Page.

## Event InitiateCheckout

InitiateCheckout sudah terpasang secara otomatis saat Anda menambahkan komponen Form Pemesanan di Landing Page Studio.

Ketika Anda menambahkan komponen ini maka Anda perlu menetapkan Produk / Bundel apa yang Anda taruh di komponen ini.

Setelah produknya Anda tetapkan maka Anda dapat mengecek event InitiateCheckout di pengaturan komponen Form Pemesanan.

Itulah default events Scalev yang telah terpasang secara otomatis.

Pengaturan default Scalev ini dapat diubah sendiri oleh Anda.

Event pixel selain dari tiga yang dibahas di sini seperti AddToCart, AddPaymentInfo, Lead, dll, dapat ditambahkan sendiri sesuai kebutuhan alur bisnis Anda.

Jika default events ini sudah cocok dengan kebutuhan Anda maka tidak perlu mengubah apapun.
