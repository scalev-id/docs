Sebelum lanjut, pastikan Anda sudah melakukan instruksi di artikel berikut:

1. [Tutorial Cara Menambahkan Pixel TikTok ke Scalev](https://tutorial.scalev.id/tambah-pixel-tiktok/ "Tutorial Cara Menambahkan Pixel TikTok ke Scalev")

3. [Tutorial Cara Memasang Event Pixel TikTok di Landing Page Scalev](https://tutorial.scalev.id/setting-pixel-tiktok/ "Tutorial Cara Memasang Event Pixel TikTok di Landing Page Scalev")

Jika sudah, Anda boleh mulai melakukan pengetesan pixel TikTok yang terpasang di LP Scalev Anda.

Ikuti langkah-langkah di bawah ini.

### 1\. Data sources

Buka halaman Events Manager TikTok.

Di menu sebelah kiri klik **Data sources**.

Pilih pixel yang akan Anda test.

<figure>

![Testing Pixel Tiktok 1](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-1-1024x596.png)

<figcaption>

Halaman depan Events Manager TikTok

</figcaption>

</figure>

### 2\. Test Server Events

Di halaman pixel yang Anda pilih, klik tab Test Events.

Cek bagian **Test Server Events** di sebelah kiri.

Salin kode testing yang muncul di Step 2.

<figure>

![Testing Pixel Tiktok 2](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-2-795x1024.png)

<figcaption>

Salin kode testing

</figcaption>

</figure>

### 3\. Kode Testing

Masuk ke dasbor Scalev, lalu buka halaman Settings → Analitik → TikTok Pixel.

Pilih pixel TikTok yang sudah Anda pasang, lalu klik titik tiga di sebelah kanannya.

Klik Lihat / Edit.

<figure>

![Testing Pixel Tiktok 3](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-3-1024x436.png)

<figcaption>

Edit pixel TikTok di pengaturan Analitik Scalev

</figcaption>

</figure>

Masukkan kode testing yang telah disalin di langkah sebelumnya.

![Testing Pixel Tiktok 4](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-4-745x1024.png)

Klik Simpan.

### 4\. Link Landing Page

Pergi ke menu Landing Page di dasbor Scalev.

Pilih dan salin link LP yang ingin Anda test.

Contoh:

demo.lpngebut.com/landing-page-baru

### 5\. Generate QR code

Kembali ke halaman Events Manager TikTok.

Di tab Test Events, paste link LP di kotak yang tersedia.

Lalu klik Generate QR Code.

<figure>

![Testing Pixel Tiktok 5](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-5-795x1024.png)

<figcaption>

Generate QR Code untuk LP TikTok Anda

</figcaption>

</figure>

Gambar QR code akan muncul.

### 6\. Scan QR Code

Scan gambar QR code tersebut dengan aplikasi TikTok di handphone Anda.

Caranya, buka aplikasi TikTok.

Cek profil akun TikTok Anda.

Klik menu ikon hamburger (garis tiga) di sebelah kanan atas.

<figure>

![Testing Pixel Tiktok 8](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-8-473x1024.jpg)

<figcaption>

Halaman profil TikTok Anda

</figcaption>

</figure>

Muncul popup lalu klik **Kode QR saya**.

<figure>

![Testing Pixel Tiktok 9](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-9-473x1024.jpg)

<figcaption>

Akses menuju scan lewat browser bawaan TikTok

</figcaption>

</figure>

Di sebelah kanan atas klik ikon scan QR.

<figure>

![Testing Pixel Tiktok 10](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-10-473x1024.jpg)

<figcaption>

Ikon Scan QR di aplikasi TikTok

</figcaption>

</figure>

Muncul layar kamera hp Anda, lalu scan QR code LP Anda yang telah digenerate di langkah ke 5.

Akan muncul halaman seperti gambar di bawah ini.

![Testing Pixel Tiktok 11](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-11-473x1024.png)

Klik tombol Buka.

### 7\. Testing Event

Setelah LP Anda terbuka di dalam aplikasi TikTok, test LP Anda sampai selesai.

Scroll sampai ke bawah.

Klik tombol yang mengarah ke halaman form (jika ada).

Isi form checkout.

Klik Submit.

### 8\. Event Activity

Jika sudah selesai testing LP Anda di langkah sebelumnya, kembali ke halaman Events Manager TikTok.

Cek di bagian Event Activity.

<figure>

![Testing Pixel Tiktok 7](https://tutorial.scalev.id/wp-content/uploads/2024/12/testing-pixel-tiktok-7-1024x555.png)

<figcaption>

Daftar event yang ketrigger saat testing

</figcaption>

</figure>

Jika muncul nama event pixel yang sudah Anda pasang di LP maka...

Selamat!

Testing event TikTok Anda sudah benar.

Dengan demikian pengaturan pixel TikTok Anda sampai sejauh ini sudah berhasil dan siap untuk digunakan beriklan di TikTok.

**NOTE**:

Apabila ada icon kuning tanda seru seperti di ss Event Activity di atas maka boleh diabaikan.

Jika ingin diperbaiki maka cek dulu peringatannya apa yang harus diperbaiki.
