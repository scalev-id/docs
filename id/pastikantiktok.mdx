Untuk memastikan agar event pixel TikTok tertrigger di Scalev.

Silahkan lakukan 6 hal ini:

## Pertama,

Pastikan sudah menghubungkan pixel tiktok di setting -> analitik -> tiktok pixel  
_notes : kode testing jangan diisi jika bukan untuk testing_

![Screen Shot 2025 01 09 At 09.39.42](https://tutorial.scalev.id/wp-content/uploads/2025/01/Screen-Shot-2025-01-09-at-09.39.42-1024x592.png)

## ⇣

## Kedua,

Pastikan dipengaturan toko sudah mengaktifkan event trigger.  
[](https://storage.3.basecamp.com/5878785/blobs/6f474fdc-ce35-11ef-a485-1ab7a0b123c9/download/Screen%20Shot%202025-01-09%20at%2009.46.59.png)

## Ketiga,

Pastikan di landing page sudah di set tiktok pixel  
[](https://storage.3.basecamp.com/5878785/blobs/8f48f48e-ce35-11ef-9248-0e9c1bc662d2/download/Screen%20Shot%202025-01-09%20at%2009.40.48.png)

## ⇣

## Keempat,

Tampilan ketika calon customer membuka landing page (pixel tiktok) pada ads manager akan muncul Event View Content

![Screen Shot 2025 01 09 At 09.59.06](https://tutorial.scalev.id/wp-content/uploads/2025/01/Screen-Shot-2025-01-09-at-09.59.06-1024x351.png)

## ⇣

## Kelima,

Tampilan ketika calon customer sudah isi form landing page (pixel tiktok) pada ads manager akan muncul Event Initiated Checkout  
[](https://storage.3.basecamp.com/5878785/blobs/0931a69c-ce36-11ef-89f7-8e49e4c60525/download/Screen%20Shot%202025-01-09%20at%2010.00.53.png)

## Keenam,
