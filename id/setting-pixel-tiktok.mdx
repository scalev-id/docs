---
title: "Tutorial Cara Memasang Event Pixel TikTok di Landing Page Scalev"
description: "Ikuti langkah-langkah berikut ini:"
---
Jika Anda sudah [menambahkan pixel baru TikTok](https://tutorial.scalev.id/tambah-pixel-tiktok/ "Tutorial Cara Menambahkan Pixel TikTok ke Scalev") maka selanjutnya Anda perlu memasang pixel dan event yang diperlukan di Landing Page Anda.

Ikuti langkah-langkah berikut ini:

### 1\. Landing Page Studio (LPS)

Masuk ke dasbor Scalev lalu pergi ke menu Landing Page dan pilih / buat LP baru.

<figure>

![Cara Setting Pixel Tiktok 1](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-setting-pixel-tiktok-1-1024x408.png)

<figcaption>

Buka LP Anda di Scalev

</figcaption>

</figure>

### 2\. Pengaturan Pixel TikTok

Di halaman LPS cek tab Pengaturan.

Lalu scroll ke bawah sampai ketemu bagian pixel TikTok.

Pilih dan pasang pixel TikTok yang sudah ditambahkan ke Scalev.

Pasang juga event yang cocok untuk LP Anda.

<figure>

![Cara Setting Pixel Tiktok 2](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-setting-pixel-tiktok-2-898x1024.png)

<figcaption>

Pengaturan pixel TikTok di LP

</figcaption>

</figure>

### 3\. Komponen Form Pemesanan

Masih di halaman LPS.

Cek tab Konten lalu edit / tambahkan komponen Form Pemesanan.

Klik ikon Gear (gerigi) untuk pergi ke pengaturan komponen Form Pemesanan.

<figure>

![Cara Setting Pixel Tiktok 3](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-setting-pixel-tiktok-3-953x1024.png)

<figcaption>

Pengaturan komponen Form Pemesanan

</figcaption>

</figure>

Scroll ke bawah sampai ketemu bagian event pixel TikTok.

Masukkan event standard TikTok, contoh: InitiateCheckout.

<figure>

![Cara Setting Pixel Tiktok 4](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-setting-pixel-tiktok-4-884x1024.png)

<figcaption>

Event TikTok di komponen Form Pemesanan

</figcaption>

</figure>

Event ini akan trigger saat form disubmit oleh visitor.

### 4\. Ulangi

Ulangi langkah-langkah di atas untuk LP aktif yang lainnya sesuai kebutuhan funnel Anda.

### 5\. Selesai.

Sampai sini Anda sudah berhasil mengatur pixel TikTok dengan event-eventnya yang terpasang di LP aktif Anda.

Selanjutnya Anda perlu melakukan pengetesan event pixel TikTok ini untuk memastikan semua pengaturan pixel TikTok sudah benar.

Klik judul artikel di bawah ini untuk panduan melakukan testing pixel TikTok.
