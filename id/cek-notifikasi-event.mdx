Untuk mencari tahu apakah event yang terjadi di Scalev ke Meta Ads sudah berhasil dikirim atau belum maka Anda bisa ikuti langkah-langkah berikut ini:

1. Masuk ke dasbor Scalev

2. Buka halaman Order → Lalu pergi ke deretan ikon sebelah kanan atas → Klik ikon kolom tiga

<Frame>
  ![Cek Notifikasi Event 1 1024x468](/images/cek-notifikasi-event-1-1024x468.png)
</Frame>

3. Muncul dialog "Atur Kolom Daftar Pesanan" lalu centang pilihan Purchase Event Triggered

<Frame>
  ![Cek Notifikasi Event 2 1024x455](/images/cek-notifikasi-event-2-1024x455.png)
</Frame>

4. Setelah disimpan selanjutnya akan muncul kolom baru "Purchase Event Triggered" di halaman Order

<Frame>
  ![Cek Notifikasi Event 3 1024x566](/images/cek-notifikasi-event-3-1024x566.png)
</Frame>

Keterangan:

- Jika ikon bulatan X di sebelah kiri Meta CAPI berwarna abu-abu maka event Purchase belum dikirimkan ke Meta
- Jika ikon bulatan centang di sebelah kiri Meta CAPI berwarna hijau maka event Purchase sudah dikirimkan ke Meta

Kenapa bulatan X masih berwarna abu-abu?

Bisa jadi karena:

- Proses pengiriman event Purchase menunggu siklus sinkronisasi server antara Scalev dengan Meta
- Status order belum Confirmed / Unpaid, atau
- Status order lainnya yang Anda tetapkan sendiri agar event Purchase ini masih belum triggered
