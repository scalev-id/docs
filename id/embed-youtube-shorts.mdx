Jika Anda ingin menampilkan video YouTube Shorts di LP Scalev Anda dapat menggunakan tool di atas.

Berbeda dengan video reguler YouTube, kode embed video Shorts tidak mudah ditampilkan oleh situs YouTube-nya langsung.

Untuk memudahkannya maka kami membuat tool di atas agar Anda dapat membuat kode embed siap pakai.

**Tool ini dibuat khusus untuk user Scalev.**

Caranya ikuti langkah-langkah berikut ini.

### 1\. Buka YouTube

Buka video Anda di situs web atau aplikasi YouTube.

1. Klik tombol Share di sebelah kanan video

3. Klik tombol Salin / Copy Link

<figure>

![Embed Yt Shorts 1](https://tutorial.scalev.id/wp-content/uploads/2024/12/embed-yt-shorts-1-1024x996.png)

<figcaption>

Salin URL YouTube Shorts

</figcaption>

</figure>

### 2\. Generate Embed Code

Gunakan tool yang tersedia di awal artikel ini.

1. Tempel link video Anda di field URL

3. Klik tombol Generate Embed Code, hasilnya muncul di dalam kotak bawah.

5. Klik ikon Copy untuk menyalin kodenya

![Embed Yt Shorts 2](https://tutorial.scalev.id/wp-content/uploads/2024/12/embed-yt-shorts-2.png "Generate embed code yt shorts")

### 3\. Komponen HTML

Buka Landing Page Studio di dasbor Scalev Anda.

Lalu tambahkan komponen HTML.

Tempelkan kode embed yang disalin tadi ke dalam komponen HTML ini.

Simpan & Terbitkan LP Anda.

![Embed Yt Shorts 3](https://tutorial.scalev.id/wp-content/uploads/2024/12/embed-yt-shorts-3-992x1024.png)

Selesai.
