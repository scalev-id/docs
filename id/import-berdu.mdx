---
title: "Export Import Customer dari Berdu ke Scalev (SEKALIGUS Ter-Import Seluruh History Orderan Si Customer Tersebut)"
description: "Masuk ke Berdu, lalu pergi ke halaman Apps → Data Exporter."
---
Masuk ke Berdu, lalu pergi ke halaman Apps → Data Exporter.

<figure>

![Impor Pelanggan Berdu 1](https://tutorial.scalev.id/wp-content/uploads/2024/08/impor-pelanggan-berdu-1-1024x750.jpeg)

<figcaption>

Buka fitur pengekspor data pelanggan Berdu dengan Data Exporter

</figcaption>

</figure>

Di halaman Data Exporter pilih Ekspor Pelanggan lalu klik Download.

Setelah itu masuk ke Scalev, lalu pergi ke halaman Customer dan klik tombol kotak panah atas.

<figure>

![Impor Pelanggan Berdu 2](https://tutorial.scalev.id/wp-content/uploads/2024/08/impor-pelanggan-berdu-2-1024x750.jpeg)

<figcaption>

Buka fitur Upload Data Pelanggan

</figcaption>

</figure>

Akan muncul dialog baru, lalu klik link Download Template.

<figure>

![Impor Pelanggan Berdu 3](https://tutorial.scalev.id/wp-content/uploads/2024/08/impor-pelanggan-berdu-3-1024x639.png)

<figcaption>

Download file template dari Scalev

</figcaption>

</figure>

Buka file customers\_template.csv yang baru didownload dari Scalev dengan aplikasi pengolah data seperti Ms. Excel atau Google Sheets.

<figure>

![Impor Pelanggan Berdu 4](https://tutorial.scalev.id/wp-content/uploads/2024/08/impor-pelanggan-berdu-4-1024x473.png)

<figcaption>

Template dokumen pelanggan Scalev

</figcaption>

</figure>

Di dalam file csv tersebut ada 4 kolom default Scalev, yaitu:

1. names

3. phone

5. emails

7. date\_of\_birth

Kemudian buka juga file csv yang telah di-download dari Berdu.

<figure>

![Impor Pelanggan Berdu 5](https://tutorial.scalev.id/wp-content/uploads/2024/08/impor-pelanggan-berdu-5-1024x331.png)

<figcaption>

Template dokumen pelanggan Berdu

</figcaption>

</figure>

Di dalam file csv Berdu biasanya ada kolom yang dapat disesuaikan dengan kolom Scalev, seperti:

1. Nama Depan

3. Nama Belakang

5. Telepon

7. Email

Salin data di kolom yang sesuai dengan kolom default Scalev, contoh: kolom bernama Nama Depan datanya di-paste ke names, lalu Telepon ke phones, dst.

Kolom emails jika tidak ada datanya boleh diabakan (tidak perlu diisi).

Setelah semua data di Berdu disalin ke file customers\_template.csv jangan lupa untuk disimpan (Save) dulu perubahannya dengan aplikasi pengolah data yang Anda gunakan.

Kembali ke halaman Customer → Upload Data Pelanggan di Scalev.

Klik tombol Choose File, pilih file customers\_template.csv lalu klik Upload

<figure>

![Impor Pelanggan Berdu 6](https://tutorial.scalev.id/wp-content/uploads/2024/08/impor-pelanggan-berdu-6.png)

<figcaption>

Upload Data Pelanggan di Scalev

</figcaption>

</figure>

Jika sudah berhasil Upload maka impor data pelanggan dari Berdu ke Scalev telah selesai.
