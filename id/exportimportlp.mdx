1. Kalau Anda sudah pernah bikin LP di account Scalev Anda, terus Anda pengen bikin LP sejenis...

3. Kan tinggal duplicate aja tuh LPnya...

5. Abis itu tinggal edit - edit deh LP hasil duplicate-annya.

​⇣ 

Nah gimana kalau LPnya pengen dipake, di account Scalev lain?

# Misal:

- Pemilik bisnis yang ingin LPnya dijadiin rujukan buat para karyawan/advertisernya temen. Berbagi ke sesama IMers. Atau mungkin mau ngasih LP rujukan ke murid sendiri. Atau mungkin mau jadiin bonus affiliate.

- Pemilik produk yang LPnya mau diiklanin sama affiliate, reseller, dropshipper-nya.

- Pemilik jasa LP, yang pengen sat set sat set ngelayanin banyak bikin LP untuk para client - clientnya.

Kan gak bisa tuh!

⇣ 

# Kabar Baiknya:

Sekarang bisa.

Dengan feature export import LP.

1. Sebuah LP, bisa diexport. Hasil export-annya berupa sebuah file txt.

3. File txtnya bisa diimport ke account Scalev yang lain. 

5. Plek ketiplek, bakalan sama percis LP-nya.

![Anjay Indah Cahya Nabila](https://tutorial.scalev.id/wp-content/uploads/2024/08/anjay-indah-cahya-nabila.gif)

\===

# Tutorial:

**1/ Export Landing Page:  
**

1. Di LP yang mau Anda export, ada titik tiga di paling kanan,

3. Klik titik tiga tersebut,

5. Pilih “Export”. Save & Beri nama filenya.

![Published Titik Tiga](https://tutorial.scalev.id/wp-content/uploads/2024/08/published-titik-tiga.png)

​.

![Export Lp](https://tutorial.scalev.id/wp-content/uploads/2024/08/export-lp.png)

\===

**2/ Import Landing Page:  
**

1. Ke menu “Landing Page”

3. Klik button “+ Tambah Landing Page”

5. Pilih yang “Import”. Pilih file LP yang mau diimport & Upload

​

![Tambah Landing Page](https://tutorial.scalev.id/wp-content/uploads/2024/08/tambah-landing-page.png)

.

![Import Dari File](https://tutorial.scalev.id/wp-content/uploads/2024/08/import-dari-file.png)

\===
