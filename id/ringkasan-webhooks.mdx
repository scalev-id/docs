---
title: 'Ringkasan webhooks'
---

Webhook adalah cara Scalev memberi tahu sistem Anda saat event tertentu terjadi. Saat event yang Anda pilih terjadi, Scalev akan mengirim request HTTP POST ke URL webhook yang Anda daftarkan. Anda dapat memakai webhook untuk menjalankan kode atau aksi khusus di sistem Anda.

Webhook dikirim sebagai request POST dengan payload JSON di request body. Payload berisi tipe event dan data yang terkait dengan event tersebut. URL webhook harus berupa endpoint publik yang dapat menerima request POST dari Scalev.
