Mekanisme pemotongan PPh Pasal 23 PT Interna Cipta Asia:

1. User Scalev dapat mengajukan pengembalian bukti potong dengan mengirim file PDF eBupot (bukti potong elektronik)

3. Untuk sampai dengan saat ini PT Interna Cipta Asia belum memiliki SKB, kami masih proses pengajuan untuk SKB

5. Dikarenakan metode pembayaran berlangganan Scalev menggunakan VA sehingga untuk metode pemotongan langsung atas PPh pasal 23 tidak bisa langsung, mekanisme pemotongan sesuai yang sudah kami jelaskan di point 1

7. Untuk KPP PT Interna Cipta Asia di KPP Pratama Jakarta Kebayoran Lama
