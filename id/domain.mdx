Anda tinggal mengganti domain bawaan Scalev menjadi Domain pribadi Anda.

Contoh:

- Yang Awalnya ⇢ https://elpe.myscalev.com/ngebut 

- Diubah menjadi ⇢ https://lpngebut.com/ngebut

**Catatan:**

1. Jika DNS type A dengan Name domain terdapat lebih dari 1, maka hapus DNS type A selain DNS type A dari Scalev. 

3. DNS Type A wajib mematikan Proxy / diubah jadi DNS Only.

5. Verifikasi domain biasanya membutuhkan waktu maksimal 2×24 jam, baik DNS Type TXT ataupun DNS Type A. Jadi jika setelah memasukan DNS ke Cloudflare ternyata belum bisa di verifikasi di Scalev, silahkan tunggu & lakukan klik tombol verifikasi secara berkala.

### Domain sudah aktif tapi tidak dapat diakses?

Ketika domain Anda sudah ditambahkan sesuai instruksi dan dipastikan sudah benar, tapi Anda tidak dapat mengaksesnya sendiri di laptop.

Padahal kalau orang lain mengakses domain Anda ternyata sudah aktif dan bisa diakses dengan normal.

Maka kendalanya terletak pada **cache jaringan internet dan browser Anda.**

Solusinya ada tiga:

1. Restart modem/wifi internet Anda

3. Hapus cache di browser

5. Flush DNS

Dengan merestart modem/wifi Anda, biasanya koneksi internet Anda mendapatkan alamat IP baru yang fresh untuk mengakses internet.

Pastikan juga cache pada browser Anda telah dihapus.

Caranya:

Di browser Chrome / Firefox, pencet tombol CTRL + SHIFT + DEL.

Centang:

- Browsing history

- Cached images and files

Time range pilih All time.

<figure>

![Delete Cache](https://tutorial.scalev.id/wp-content/uploads/2024/03/delete-cache-1024x954.png)

<figcaption>

Hapus cache browser

</figcaption>

</figure>

Lalu klik tombol Delete Data.

Untuk menghapus cache koneksi internet di laptop Anda ikuti cara berikut ini.

**Pengguna Windows:**

1. Buka aplikasi CMD dengan memencet kombinasi tombol logo Windows WIN + R

3. Ketik `ipconfig /flushdns`

5. Tekan Enter

7. Selesai

**Pengguna Mac:**

1. Buka aplikasi Terminal dengan memencet CMD + SPACE

3. Ketik terminal, tekan Enter

5. Jendela Terminal terbuka, ketik `dscacheutil -flushcache`

7. Tekan Return (Enter)

9. Selesai

Demikian solusi domain aktif yang tidak dapat diakses di laptop Anda setelah menambahkan domain baru di Scalev.

Jika Anda sudah mencoba seluruh solusi di atas namun tidak juga berhasil, silakan hubungi CS Scalev.
