Kenapa skornya berubah-ubah?

1\. Skor ini dicek melalui random network milik Google. Terkadang ada network mereka yang tidak "ramah" dengan ISP di Indonesia. Tapi umumnya aman.

2\. Komponen terbesar yang diload oleh web umumnya ada 2: pixel dan image.

Seringkali pixel Meta unstable, alias mogok di tengah jalan. Kalau ini kasusnya sudah wallahualam. Cara mengeceknya, kadang di pagespeed muncul seperti ini:

![Faq Pagespeed Min](https://tutorial.scalev.id/wp-content/uploads/2024/09/faq-pagespeed-min.png)

Ini kasusnya pixel mogok alias gagal dieksekusi.

Kalau kasusnya adalah image, maka Anda tinggal mengikuti tutorial saja untuk melakukan treatment terhadap image.
