Untuk menambahkan pixel (sekarang namanya Datasets di Meta) ke Scalev silakan ikuti langkah-langkah berikut ini:

1\. Pertama Anda buka halaman Business Manager lalu pergi ke Business Settings → Data sources → Datasets → Klik pixel ID di dekat nama pixel Anda

<figure>

![Tambah pixel update](https://tutorial.scalev.id/wp-content/uploads/2024/07/tambah-pixel-update-1024x640.png)

<figcaption>

Salin kode pixel Anda

</figcaption>

</figure>

2\. Buka dasbor Scalev lalu pergi ke Setting → Analitik → Klik + Tambah Analitik, maka akan muncul dialog seperti ini:

<figure>

![Tambah Pixel Scalev 3](https://tutorial.scalev.id/wp-content/uploads/2024/07/tambah-pixel-scalev-3-761x1024.png)

<figcaption>

Masukkan nama dan kode pixel di kolom berbintang merah

</figcaption>

</figure>

3\. Beri nama pixel yang Anda buat lalu paste pixel-nya di kolom Pixel ID setelah itu klik tombol Simpan

4\. Setelah berhasil disimpan kira - kira tampilannya akan seperti ini (dalam case ini, sudah menambahkan banyak pixel ID ke dalam account Scalev):

![Tambah Pixel Scalev 4](https://tutorial.scalev.id/wp-content/uploads/2024/07/tambah-pixel-scalev-4-1024x488.png)

Selanjutnya untuk menambahkan Conversion API di pengaturan pixel Scalev silakan Anda buka halaman tutorial ini → [Cara Mengambil Conversion API Token Access](https://tutorial.scalev.id/conversionapi/)
