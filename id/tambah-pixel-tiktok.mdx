Untuk menambahkan pixel TikTok ke Scalev ikuti langkah-langkah berikut ini sampai selesai:

### 1\. Login ke akun iklan TikTok

Dengan akses Business Center yang Anda miliki.

Klik link halaman login ini: [https://ads.tiktok.com/i18n/login](https://ads.tiktok.com/i18n/login)

<figure>

![Cara Menambahkan Pixel Tiktok 1](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-1-826x1024.png)

<figcaption>

Halaman login ke akun iklan TikTok

</figcaption>

</figure>

### 2\. Pilih akun iklan aktif yang akan digunakan

Setelah masuk ke halaman Business Center klik menu Accounts → All Accounts → Pilih salah satu iklan aktif Anda

### 3\. Accounts

Klik View di kolom Action sebelah kanan

<figure>

![Cara Menambahkan Pixel Tiktok 2](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-2-1024x641.png)

<figcaption>

Pilih akun iklan untuk membuat pixel baru

</figcaption>

</figure>

### 4\. Ads Manager

Klik tombol Ads Manager untuk membuka halaman akun iklan tersebut.

<figure>

![Cara Menambahkan Pixel Tiktok 3](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-3-1024x544.png)

<figcaption>

Tombol untuk pergi ke Ads Manager

</figcaption>

</figure>

### 5\. Events

Di halaman Ads Manager klik menu Tools → Events

<figure>

![Cara Menambahkan Pixel Tiktok 4](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-4-1024x545.png)

<figcaption>

Halaman akun iklan TikTok

</figcaption>

</figure>

### 6\. Connect data source

Di halaman Events Manager klik tombol **\+ Connect data source**

<figure>

![Cara Menambahkan Pixel Tiktok 5](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-5-902x1024.png)

<figcaption>

Tombol untuk membuat pixel baru TikTok

</figcaption>

</figure>

### 7\. Add your website

Masukkan alamat website Anda.

Contoh:

demo.lpngebut.com

Masukkan tanpa prefix **https://** karena sudah terisi otomatis.

Klik Next.

<figure>

![Cara Menambahkan Pixel Tiktok 6](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-6-1024x704.png)

<figcaption>

Popup pengisian alamat website

</figcaption>

</figure>

### 8\. Choose data connection method

Pilih **Manual setup** lalu klik tombol Next.

<figure>

![Cara Menambahkan Pixel Tiktok 7](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-7-1024x738.png)

<figcaption>

Popup metode pixel

</figcaption>

</figure>

### 9\. Choose a manual connection method

Pilih yang direkomendasikan oleh TikTok, yaitu: TikTok Pixel + Events API.

Klik Next.

<figure>

![Cara Menambahkan Pixel Tiktok 8](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-8-1024x796.png)

<figcaption>

Popup pilihan jenis pixel TikTok

</figcaption>

</figure>

### 10\. Create pixel

Beri nama pixel Anda.

Nama bebas.

Contoh:

Demo Pixel TikTok

<figure>

![Cara Menambahkan Pixel Tiktok 9](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-9-1024x730.png)

<figcaption>

Popup nama pixel baru

</figcaption>

</figure>

### 11\. Manage configurations

Perhatikan langkah sebelumnya di langkah ke 2 "Install base code" itu lewati saja.

Langsung ke langkah ke 3.

Aktifkan **Enable Automatic Advanced Matching (Recommended)**.

<figure>

![Cara Menambahkan Pixel Tiktok 10](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-10-1024x744.png)

<figcaption>

Popup Enable Automatic Advanced Matching

</figcaption>

</figure>

### 12\. Implement Events API

Perhatikan langkah sebelumnya di langkah ke 4, ke 5, ke 6, itu dilewati saja dengan cara klik Next terus.

Langkah ke 7 klik tombol **Generate access token.**

<figure>

![Cara Menambahkan Pixel Tiktok 11](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-11-1024x742.png)

<figcaption>

Popup generate access token

</figcaption>

</figure>

Setelah diklik maka akan muncul kode token akses seperti ini:

<figure>

![Cara Menambahkan Pixel Tiktok 12](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-12-1024x399.png)

<figcaption>

Kode access token

</figcaption>

</figure>

Simpan dan amankan dulu kode tersebut di mana aja.

Notepad contohnya.

Jangan lupa klik Next.

### 13\. Testing

Untuk tahap testing Anda boleh lewati aja dulu.

Nanti akan dibuatkan tutorial khusus untuk testing pixel TikTok ini.

Klik tombol Finish.

<figure>

![Cara Menambahkan Pixel Tiktok 10a](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-10a-1024x728.png)

<figcaption>

Popup Testing pixel TikTok

</figcaption>

</figure>

### 14\. Data sources

Kembali ke halaman depan Events Manager.

Klik Data sources.

Salin kode pixel TikTok Anda yang baru.

Simpan dan amankan ke mana aja.

Ke Notepad contohnya.

![Cara Menambahkan Pixel Tiktok 13](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-13-1024x629.png)

Salin kode pixel TikTok Anda

### 15\. Settings

Masuk ke dasbor Scalev lalu pergi ke menu Settings → Analitik.

Pilih TikTok Pixel di dropdown Analitik.

Lalu klik **\+ Tambah Analitik**

<figure>

![Cara Menambahkan Pixel Tiktok 14](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-14-1024x505.png)

<figcaption>

Tambah Analitik TikTok Pixel

</figcaption>

</figure>

### 16\. Tambah Analitik

Muncul popup seperti di bawah ini.

Isi rincian data pixel yang tadi telah dibuat di Business Center TikTok

<figure>

![Cara Menambahkan Pixel Tiktok 15](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-15-730x1024.png)

<figcaption>

Popup pengisian data pixel baru TikTok

</figcaption>

</figure>

Jangan lupa klik tombol Simpan.

### 17\. Selesai

Jika sudah berhasil ditambahkan maka tampilannya akan seperti ini.

<figure>

![Cara Menambahkan Pixel Tiktok 16](https://tutorial.scalev.id/wp-content/uploads/2024/12/cara-menambahkan-pixel-tiktok-16-1024x399.png)

<figcaption>

Pixel TikTok berhasil ditambahkan ke Scalev

</figcaption>

</figure>

**NOTE:**

Perlu diingat bahwa penambahan pixel baru TikTok tersebut walaupun sudah selesai tapi **BELUM DIPASANG** secara otomatis di Landing Page Anda.

Sehingga belum siap untuk digunakan.
